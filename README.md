# Actividad de Seguimiento - Estructuras de Datos

Este repositorio contiene la implementación de varias estructuras de datos lineales en **Java**, desarrolladas desde cero utilizando la clase **Nodo** y clases específicas para cada estructura.

El proyecto fue realizado como parte de una **actividad de seguimiento de la asignatura Estructuras de Datos**, con el propósito de reforzar los conceptos fundamentales de representación, inserción, eliminación, búsqueda y recorrido de estructuras dinámicas.

## Integrantes

- **Jose Manuel Rodriguez Ospina**
- **Kevin Andres Lopez Ramirez**

## Objetivo

Desarrollar e implementar diferentes estructuras de datos en Java, construyéndolas manualmente a partir de nodos enlazados, sin usar colecciones prediseñadas del lenguaje, con el fin de comprender su funcionamiento interno.

## Estructuras implementadas

En este proyecto se incluyen las siguientes estructuras:

- **Lista simplemente enlazada**
- **Lista doblemente enlazada**
- **Lista circular simplemente enlazada**
- **Cola**
- **Bicola**
- **Cola de prioridad**
- **Pila**

## Funcionalidades principales

Dependiendo de la estructura, se implementan operaciones como:

- Agregar elementos
- Eliminar elementos
- Buscar elementos
- Mostrar elementos
- Recorrer la estructura

## Ejecución del proyecto

1. Clonar este repositorio:

        git clone <URL_DEL_REPOSITORIO>

2. Abrir el proyecto en un IDE compatible con Java, por ejemplo:
   - IntelliJ IDEA
   - Eclipse
   - Visual Studio Code

3. Verificar que la estructura de paquetes coincida con la declaración:

        package co.edu.uniquindio;

4. Ejecutar la clase `Main.java`.
